# 🎵 Thư mục Nhạc

Đặt các file nhạc .mp3 vào đây với tên:
- `happy-birthday-1.mp3`  → Bài hát 1 (Happy Birthday To You)
- `happy-birthday-2.mp3`  → Bài hát 2 (Chúc Mừng Sinh Nhật)
- `happy-birthday-3.mp3`  → Bài hát 3 (Birthday Melody)

Để thêm hoặc đổi tên bài hát, chỉnh sửa trong:
`app/Http/Controllers/BirthdayController.php` → mảng `$songs`
